# pyapi-gitlab

pyapi-gitlab is a python wrapper for the [Gitlab API](https://github.com/gitlabhq/gitlabhq/tree/master/doc/api).

## Requirements

- requests
- markdown

## Installation

pyapi-gitlab is now on Pypi! Install it with:

```bash
pip install pyapi-gitlab
```
